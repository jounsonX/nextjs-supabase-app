# Task 005: 인증/프로필 및 모임 목록·생성 UI 구현

## 개요

Phase 2 UI/UX 완성 단계. 홈·모임 목록·모임 생성·프로필·인증 페이지 UI를 더미 데이터로 완성한다.

실제 Supabase 연동은 Phase 3(Task 008~011)에서 수행한다.
