# 소규모 모임 이벤트 관리 MVP PRD

## 핵심 정보

**목적**: 소규모 정기 모임 주최자가 공지·참여자 관리·정산·카풀을 카카오톡 단체방 대신 한 곳에서 처리할 수 있도록 한다.
**사용자**: 수영/헬스 정기 운동, 친구 소셜 모임, 소규모 동호회를 주관하는 5~15인 규모 모임의 주최자 및 참여자

---

## 사용자 여정

```
1. 랜딩(홈) 페이지
   ↓ 로그인 / 회원가입 버튼 클릭

2. 로그인 페이지 ─── 이메일/비밀번호 또는 Google OAuth
   ↓ 인증 성공 → 자동 리디렉션

3. 내 모임 대시보드 (보호 라우트)
   ↓ 탭 선택 분기
   [내가 주최한 모임] → 모임 목록 페이지 (주최자 탭)
   [내가 참여한 모임] → 모임 목록 페이지 (참여자 탭)
   [새 모임 만들기 버튼 클릭] → 모임 생성 페이지

4. 모임 생성 페이지
   ↓ 저장 성공 → 모임 상세 페이지 (자동 이동)

5. 모임 상세 페이지 (탭 구조)
   ├── [정보] 탭      → 모임 정보 조회, 참여 신청/취소
   ├── [공지] 탭      → 공지 목록 조회 / 공지 작성 페이지로 이동 (주최자)
   ├── [정산] 탭      → 정산 현황 조회, 완료 체크 토글 (주최자)
   └── [카풀] 탭      → 카풀 목록, 카풀 등록/신청

   [수정 버튼] (주최자) → 모임 수정 페이지
   ↓ 공지 작성 버튼 클릭 (주최자)

6. 공지 작성 페이지
   ↓ 저장 성공 → 모임 상세 페이지 [공지] 탭 (자동 이동)

7. 완료
   ↓ 다음 액션 선택
   → 모임 목록으로 돌아가기
   → 다른 탭 탐색
   → 로그아웃
```

## 기능 명세

### 1. MVP 핵심 기능

| ID       | 기능명           | 설명                                                                             | MVP 필수 이유                | 관련 페이지                        |
| -------- | ---------------- | -------------------------------------------------------------------------------- | ---------------------------- | ---------------------------------- |
| **F001** | 모임 생성        | 제목, 날짜/시간, 장소, 정원(선택), 비용, 설명 입력 후 모임 개설                  | 서비스 핵심 객체 생성        | 모임 생성 페이지                   |
| **F002** | 모임 목록 조회   | 내가 주최한 모임 / 내가 참여한 모임 탭 분리 조회                                 | 모임 탐색의 출발점           | 내 모임 대시보드, 모임 목록 페이지 |
| **F003** | 모임 상세 조회   | 모임 정보·공지·정산·카풀을 탭 구조로 조회                                        | 모든 기능의 진입 허브        | 모임 상세 페이지                   |

| **F005** | 공지 작성/삭제   | 주최자만 공지 작성·삭제, 핀 고정/해제                                            | 모임 운영 핵심 커뮤니케이션  | 공지 작성 페이지, 모임 상세 페이지 |
| **F006** | 공지 목록 조회   | 참여자 전체가 핀 고정 공지 우선으로 공지 조회                                    | 공지 수신은 참여자 기본 니즈 | 모임 상세 페이지                   |
| **F007** | 참여 신청/취소   | 일반 사용자가 모임 참여 신청 및 취소, 정원 초과 시 대기 자동 처리                | 모임 구성의 핵심 흐름        | 모임 상세 페이지                   |
| **F008** | 참여자 승인/거절 | 주최자가 pending 상태 참여자를 승인(approved) 또는 거절(rejected) 처리           | 모임 멤버 관리               | 모임 상세 페이지                   |
| **F009** | 정산 관리        | 총 비용 기준 1인당 금액 자동 계산, 참여자별 정산 완료 체크 (주최자 토글)         | 모임 비용 처리 핵심          | 모임 상세 페이지                   |
| **F010** | 카풀 등록/신청   | 운전자가 카풀 등록 (출발지, 동승 인원, 메모), 참여자가 선착순 자동 수락으로 신청 | 모임 이동 편의 지원          | 모임 상세 페이지                   |

## 페이지별 상세 기능

### 내 모임 대시보드

> **구현 기능:** `F002` | **메뉴 위치:** 로그인 후 기본 랜딩 (`/protected`)

| 항목            | 내용                                                                                                                                                                                                                             |
| --------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **역할**        | 로그인 직후 진입하는 모임 현황 요약 허브                                                                                                                                                                                         |
| **진입 경로**   | 로그인 성공 후 자동 리디렉션, 헤더 "대시보드" 링크 클릭                                                                                                                                                                          |
| **사용자 행동** | 내가 주최한 모임과 참여한 모임 현황을 한눈에 파악하고 모임 상세로 진입한다                                                                                                                                                       |
| **주요 기능**   | - 내가 주최한 모임 최근 N개 카드 표시 (제목, 날짜, 참여자 수)<br>- 내가 참여한 모임 최근 N개 카드 표시 (제목, 날짜, 참여 상태)<br>- 각 카드 클릭 시 모임 상세 페이지로 이동<br>- **새 모임 만들기** 버튼 (모임 생성 페이지 이동) |
| **다음 이동**   | 카드 클릭 → 모임 상세 페이지, 새 모임 만들기 → 모임 생성 페이지                                                                                                                                                                  |

### 모임 목록 페이지

> **구현 기능:** `F002` | **메뉴 위치:** 내비게이션 "모임 목록" (`/protected/events`)

| 항목            | 내용                                                                                                                                                                                                                      |
| --------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **역할**        | 내가 주최하거나 참여한 모든 모임을 탭으로 분리하여 조회                                                                                                                                                                   |
| **진입 경로**   | 헤더 "모임 목록" 메뉴 클릭, 대시보드 카드 "전체 보기" 클릭                                                                                                                                                                |
| **사용자 행동** | 탭을 전환하며 주최/참여 모임 목록을 확인하고 특정 모임 상세로 진입한다                                                                                                                                                    |
| **주요 기능**   | - [내가 주최한 모임] / [내가 참여한 모임] 탭 전환<br>- 모임 카드: 제목, 날짜/시간, 장소, 참여자 수, 모임 상태(open/closed/cancelled)<br>- 탭별 빈 상태 메시지 표시<br>- **새 모임 만들기** 버튼 (주최자 탭에서 강조 표시) |
| **다음 이동**   | 카드 클릭 → 모임 상세 페이지, 새 모임 만들기 버튼 → 모임 생성 페이지                                                                                                                                                      |

### 모임 생성 페이지

> **구현 기능:** `F001` | **메뉴 위치:** "새 모임 만들기" 버튼 (`/protected/events/new`)

| 항목            | 내용                                                                                                                                                                                                                                                                                                       |
| --------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **역할**        | 새 모임(이벤트) 개설 전용 폼 페이지                                                                                                                                                                                                                                                                        |
| **진입 경로**   | 대시보드 또는 모임 목록 페이지의 "새 모임 만들기" 버튼 클릭                                                                                                                                                                                                                                                |
| **사용자 행동** | 모임 정보를 입력하고 저장하여 새 모임을 개설한다                                                                                                                                                                                                                                                           |
| **주요 기능**   | - 필수 입력: 제목(텍스트), 날짜/시간(datetime-local), 모임 상태(open/closed)<br>- 선택 입력: 장소(텍스트), 정원(숫자, 미입력 시 무제한), 비용(숫자 원 단위), 설명(텍스트에어리어)<br>- Server Action 기반 폼 제출 (`useActionState`)<br>- 유효성 검사: 제목 필수, 날짜 과거 불가<br>- **모임 만들기** 버튼 |
| **다음 이동**   | 저장 성공 → 생성된 모임의 모임 상세 페이지 자동 이동, 실패 → 인라인 에러 메시지 표시                                                                                                                                                                                                                       |
