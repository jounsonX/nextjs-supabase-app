# MeetUp Manager MVP PRD

## 핵심 정보

**목적**: 소규모 친목 모임(5~20명)에서 주최자가 공지·참여자 관리·카풀·정산을 카카오톡 단체방 없이 한 곳에서 처리할 수 있도록 한다.
**사용자**: 수영/헬스/친구 소셜 모임을 주관하거나 참여하는 5~20인 규모 모임의 주최자·참여자·플랫폼 운영자

---

## 사용자 여정

```
1. 로그인 페이지
   ↓ 이메일/비밀번호 또는 Google·카카오 소셜 로그인

2. 인증 성공 → 역할 확인 후 자동 리디렉션
   [Admin]  → Admin 대시보드 페이지
   [Host]   → 홈 페이지 (Host/Member 공용)
   [Member] → 홈 페이지 (Host/Member 공용)

── Host/Member 여정 ──────────────────────────────────

3. 홈 페이지
   ↓ 하단 탭 또는 FAB 선택
   [모임 목록 탭] → 모임 목록 페이지
   [FAB 클릭]    → 모임 생성 페이지

4. 모임 생성 페이지 (Host)
   ↓ 저장 성공 → 모임 상세 페이지 (자동 이동)

5. 모임 상세 페이지 (탭 구조)
   ├── [정보] 탭  → 모임 정보 조회, 참여 신청/취소, 참여자 목록
   ├── [공지] 탭  → 공지 목록 조회 / 공지 작성 페이지로 이동 (Host)
   ├── [정산] 탭  → 1인당 금액 조회, 정산 완료 체크 (Host)
   └── [카풀] 탭  → 카풀 목록, 카풀 등록/신청

   [수정 버튼] (Host) → 모임 수정 페이지
   [공지 작성 버튼] (Host) → 공지 작성 페이지

6. 완료 → 하단 탭 또는 뒤로가기로 모임 목록 복귀

── Admin 여정 ──────────────────────────────────────

3. Admin 대시보드 페이지
   ↓ 사이드바 메뉴 선택
   [사용자 관리] → Admin 사용자 관리 페이지
   [모임 관리]   → Admin 모임 관리 페이지

4. 완료 → 사이드바로 다른 관리 페이지 이동
```

## 기능 명세

| -------- | ------------------- | ----------------------------------------------------------- | ---------------------------- | ---------------------------------- |
| **F001** | 모임 생성           | 제목·날짜/시간·장소·정원·비용·설명 입력 후 모임 개설        | 서비스 핵심 객체 생성        | 모임 생성 페이지                   |
| **F002** | 모임 목록 조회      | 내가 주최한 모임·참여한 모임 탭 분리 조회 및 카드 표시      | 모임 탐색의 출발점           | 홈 페이지, 모임 목록 페이지        |

| **F004** | 모임 수정/삭제      | Host만 모임 정보 수정 및 삭제 가능                          | 모임 관리 기본 권한          | 모임 수정 페이지, 모임 상세 페이지 |
| **F005** | 공지 작성/관리      | Host만 공지 작성·삭제, 핀 고정/해제 가능                    | 모임 운영 핵심 커뮤니케이션  | 공지 작성 페이지, 모임 상세 페이지 |
| **F006** | 공지 목록 조회      | 전체 참여자가 핀 고정 공지 우선으로 공지 목록 조회          | 공지 수신은 참여자 기본 니즈 | 모임 상세 페이지                   |
| **F007** | 참여 신청/취소      | Member가 모임 참여 신청·취소, 정원 초과 시 대기자 자동 처리 | 모임 구성의 핵심 흐름        | 모임 상세 페이지                   |
| **F008** | 참여자 승인/거절    | Host가 pending 참여자 승인·거절, 취소 시 waitlist 자동 승급 | 모임 멤버 관리               | 모임 상세 페이지                   |
| **F009** | 정산 관리           | 총 비용 기준 1/n 자동 계산(올림), 참여자별 정산 완료 체크   | 모임 비용 처리 핵심          | 모임 상세 페이지                   |
| **F010** | 카풀 매칭           | 드라이버 카풀 등록(출발지·좌석), 참여자 선착순 동승 신청    | 모임 이동 편의 지원          | 모임 상세 페이지                   |
| **F013** | Admin 통계 대시보드 | 총 모임 수·신규·활성·정산 완료율 등 플랫폼 전체 통계 조회   | 플랫폼 운영 현황 파악        | Admin 대시보드 페이지              |
| **F014** | 사용자 관리         | 가입자 목록 조회, 역할(Host/Member/Admin) 변경              | 플랫폼 사용자 통제           | Admin 사용자 관리 페이지           |
| **F015** | 모임 강제 관리      | Admin이 모임 강제 닫기·취소·삭제 처리                       | 부적절 모임 운영 통제        | Admin 모임 관리 페이지             |

## 메뉴 구조

```
Host/Member 내비게이션 (하단 탭 + FAB, 모바일 중심)
├── 홈 (하단 탭 1)
│   └── 기능: F002 (내가 주최/참여한 모임 요약 카드)
├── 모임 목록 (하단 탭 2)
│   └── 기능: F002 (주최/참여 탭 분리 전체 목록)
├── 프로필 (하단 탭 3)
│   └── 기능: F012 (표시 이름 조회·수정), F011 (로그아웃)
└── + FAB (플로팅 액션 버튼)
    └── 기능: F001 (모임 생성 페이지 이동)

모임 상세 페이지 탭 메뉴 (모임 상세 페이지 내부)
├── [정보] 탭
│   └── 기능: F003 (모임 기본 정보), F007 (참여 신청/취소), F008 (Host: 승인/거절)
├── [공지] 탭
│   └── 기능: F006 (공지 목록 조회), F005 (Host: 공지 작성 이동·삭제·핀 고정)
├── [정산] 탭
│   └── 기능: F009 (1인당 금액 계산, 정산 완료 체크)
└── [카풀] 탭
    └── 기능: F010 (카풀 등록, 신청, 남은 자리 표시)

Host 전용 메뉴 (모임 상세 페이지 내)
├── 모임 수정 버튼 - F004
└── 모임 삭제 버튼 - F004

인증 메뉴 (비로그인 사용자)
├── 로그인 - F011
└── 회원가입 - F011

Admin 내비게이션 (사이드바, 데스크톱 중심)
├── 대시보드 (사이드바 1)
│   └── 기능: F013 (플랫폼 전체 통계)
├── 사용자 관리 (사이드바 2)
│   └── 기능: F014 (가입자 목록, 역할 변경)
└── 모임 관리 (사이드바 3)
    └── 기능: F015 (모임 강제 닫기/취소/삭제)
```

## 페이지별 상세 기능

### 로그인 페이지

> **구현 기능:** `F011` | **접근:** 비로그인 사용자 전용

| 항목            | 내용                                                                                                                                                                       |
| --------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **역할**        | 이메일/비밀번호 및 소셜(Google·카카오) 로그인 전용 페이지                                                                                                                  |
| **진입 경로**   | 앱 최초 접근, 보호 라우트 접근 시 미들웨어 자동 리디렉션                                                                                                                   |
| **사용자 행동** | 이메일/비밀번호를 입력하거나 소셜 버튼으로 로그인한다                                                                                                                      |
| **주요 기능**   | - 이메일·비밀번호 로그인 폼 (유효성 검사, 에러 인라인 표시)<br>- Google OAuth 로그인 버튼<br>- 카카오 소셜 로그인 버튼<br>- 회원가입 페이지 이동 링크<br>- **로그인** 버튼 |
| **다음 이동**   | 성공 → 역할별 자동 리디렉션 (Admin: Admin 대시보드 페이지, Host/Member: 홈 페이지), 실패 → 인라인 에러 표시                                                                |

---

### 회원가입 페이지

> **구현 기능:** `F011` | **접근:** 비로그인 사용자 전용

| 항목            | 내용                                                                                                                                                              |
| --------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **역할**        | 신규 사용자 계정 생성 전용 페이지                                                                                                                                 |
| **진입 경로**   | 로그인 페이지의 "회원가입" 링크 클릭                                                                                                                              |
| **사용자 행동** | 이메일과 비밀번호를 입력하거나 소셜 계정으로 계정을 생성한다                                                                                                      |
| **주요 기능**   | - 이메일·비밀번호 입력 폼 (비밀번호 확인 포함)<br>- Google OAuth 회원가입 버튼<br>- 카카오 소셜 회원가입 버튼<br>- 이메일 인증 안내 메시지<br>- **회원가입** 버튼 |
| **다음 이동**   | 성공 → 이메일 인증 안내 후 로그인 페이지, 실패 → 인라인 에러 표시                                                                                                 |

---

### 홈 페이지

> **구현 기능:** `F002` | **메뉴 위치:** 하단 탭 1번 (홈)

| 항목            | 내용                                                                                                                                                                                                                                                                                              |
| --------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **역할**        | 로그인 직후 진입하는 모임 현황 요약 허브 (Host/Member 공용)                                                                                                                                                                                                                                       |
| **진입 경로**   | 로그인 성공 후 자동 리디렉션 (Host/Member), 하단 탭 "홈" 클릭                                                                                                                                                                                                                                     |
| **사용자 행동** | 내가 주최/참여한 모임 최근 현황을 카드로 파악하고 모임 상세로 진입한다                                                                                                                                                                                                                            |
| **주요 기능**   | - 내가 주최한 모임 최근 N개 이미지 중심 카드 (제목·날짜·참여자 수·상태 배지)<br>- 내가 참여한 모임 최근 N개 이미지 중심 카드 (제목·날짜·참여 상태 배지)<br>- 각 카드 클릭 시 모임 상세 페이지로 이동<br>- 초록 계열 브랜드 컬러, 미니멀 카드 레이아웃<br>- **+ FAB** 버튼 (모임 생성 페이지 이동) |
| **다음 이동**   | 카드 클릭 → 모임 상세 페이지, FAB → 모임 생성 페이지, 탭 클릭 → 모임 목록 페이지·프로필 페이지                                                                                                                                                                                                    |

### 모임 목록 페이지

> **구현 기능:** `F002` | **메뉴 위치:** 하단 탭 2번 (모임 목록)

| 항목            | 내용                                                                                                                                                                                                                              |
| --------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **역할**        | 내가 주최하거나 참여한 모든 모임을 탭으로 분리하여 전체 조회                                                                                                                                                                      |
| **진입 경로**   | 하단 탭 "모임 목록" 클릭, 홈 페이지 "전체 보기" 링크 클릭                                                                                                                                                                         |
| **사용자 행동** | 탭을 전환하며 주최/참여 모임 목록을 확인하고 특정 모임 상세로 진입한다                                                                                                                                                            |
| **주요 기능**   | - [내가 주최한 모임] / [내가 참여한 모임] 탭 전환<br>- 모임 카드: 이미지, 제목, 날짜/시간, 장소, 참여자 수, 모임 상태 배지 (open/closed/cancelled)<br>- 탭별 빈 상태 안내 메시지 표시<br>- **+ FAB** 버튼 (모임 생성 페이지 이동) |
| **다음 이동**   | 카드 클릭 → 모임 상세 페이지, FAB → 모임 생성 페이지                                                                                                                                                                              |

### 모임 생성 페이지

> **구현 기능:** `F001` | **메뉴 위치:** FAB 버튼 클릭 (Host 전용)

| 항목            | 내용                                                                                                                                                                                                                                                                                                           |
| --------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **역할**        | Host가 새 모임(이벤트)을 개설하는 전용 폼 페이지                                                                                                                                                                                                                                                               |
| **진입 경로**   | 홈 페이지 또는 모임 목록 페이지의 FAB (+) 버튼 클릭                                                                                                                                                                                                                                                            |
| **사용자 행동** | 모임 정보를 입력하고 저장하여 새 모임을 개설한다                                                                                                                                                                                                                                                               |
| **주요 기능**   | - 필수 입력: 제목(텍스트), 날짜/시간(datetime-local), 모임 상태(open/closed)<br>- 선택 입력: 장소(텍스트), 정원(숫자, 미입력 시 무제한), 비용(숫자, 원 단위), 설명(텍스트에어리어)<br>- Server Action 기반 폼 제출 (useActionState 패턴)<br>- 유효성 검사: 제목 필수, 날짜 과거 불가<br>- **모임 만들기** 버튼 |
| **다음 이동**   | 저장 성공 → 생성된 모임의 모임 상세 페이지 자동 이동, 실패 → 인라인 에러 메시지 표시                                                                                                                                                                                                                           |

| **주요 기능**   | **[정보] 탭** (F003, F007, F008)<br>- 모임 제목·날짜/시간·장소·정원·비용·설명 표시<br>- 참여 신청 버튼 (미신청 사용자) / 참여 취소 버튼 (신청·승인 상태)<br>- 참여 상태 배지 표시 (pending/approved/waitlisted)<br>- 참여자 목록 (Host: 승인·거절 버튼 포함, Member: 이름·상태만 표시)<br>- Host 전용: 수정 버튼 (모임 수정 페이지 이동), 삭제 버튼<br><br>**[공지] 탭** (F005, F006)<br>- 핀 고정 공지 상단 표시 후 최신순 목록<br>- 각 공지: 제목·내용 미리보기·작성일<br>- Host 전용: 공지 작성 버튼 (공지 작성 페이지 이동), 핀 토글 버튼, 삭제 버튼<br><br>**[정산] 탭** (F009)<br>- 총 비용 및 1인당 금액 (Math.ceil(cost / approved 참여자 수)) 표시<br>- 참여자별 정산 완료 체크박스 (Host만 토글 가능, Member는 읽기 전용)<br>- 정산 현황 요약: "완료 N명 / 미완료 M명"<br><br>**[카풀] 탭** (F010)<br>- 카풀 목록: 운전자명·출발지·남은 자리·메모<br>- 카풀 등록 버튼 (approved 참여자, 1인 1카풀 제한)<br>- 카풀 신청 버튼 (남은 자리 > 0, 선착순 자동 수락)<br>- 내가 신청한 카풀 하이라이트 표시 |
| **다음 이동**   | 수정 버튼 → 모임 수정 페이지, 공지 작성 버튼 → 공지 작성 페이지, 삭제 성공 → 모임 목록 페이지                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
